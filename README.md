# my_okhttp_utils
 okhttp工具类

Step 1. Add it in your root build.gradle at the end of repositories:
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

Step 2. Add the dependency
```
dependencies {
    implementation 'com.github.SmallBenniao:my_okhttp_utils:0.0.1'
}
```
